# feyntensor – A Feynman integrator for tensors

The aim of `feyntensor` is to perform integration for tensors of uniform index size (hypercube-arrays of same size in each direction). The documentation is attached as `feyntensor_UserGuide_v.0.0.1`. A jupyter-notebook with examples is also provided.
